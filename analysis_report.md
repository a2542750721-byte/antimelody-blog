# HeoWeb 项目分析报告

通过对 `HeoWeb-main` 项目的深入分析，我整理了以下值得借鉴的设计元素、技术实现方案以及视觉亮点。

## 1. 核心设计体系 (Design System)

项目通过一套严谨的 CSS 变量系统（Design Tokens）实现了高度一致的视觉风格：

-   **色彩系统**: 使用 `--heo-theme` 系列变量，方便一键更换主题色。
-   **阴影与边框**: 定义了多级阴影 (`--heo-shadow-main`, `--heo-shadow-border`)，营造出轻盈的悬浮感。
-   **磨砂玻璃效果**: 导航栏通过 `backdrop-filter: blur(10px)` 实现了现代感十足的遮罩效果。

## 2. 动效与交互亮点

这些微动效是提升网页“高级感”的关键：

-   **悬浮动画 (Floating)**: 主页 Banner 使用了 `@keyframes floating`，让图片有规律地上下浮动，增加页面灵动感。
-   **视差滚动 (Parallax)**: 引入了 `simpleParallax.js`，使背景图片或装饰图在滚动时产生位移差，提升视觉深度。
-   **平滑过渡**: 所有的悬停效果（Hover）都配置了 `transition: ease-in-out 0.3s`，交互反馈非常自然。
-   **Banner 入场动画**: 页面加载时，Banner 文字和背景有特定的出场动画 (`showbanner`)。

## 3. 布局与组件设计

-   **英雄区 (Hero Section)**: 大字标题配合悬浮 Banner 图片，视觉冲击力强。
-   **产品/项目卡片**: 采用圆角卡片流式布局，配合清晰的获取按钮 (Linkbutton) 和买断制标签 (Tips)，非常适合展示作品集。
-   **个人简介卡片**: 简洁的头像、名称和头衔组合，适合放在侧边栏或主页底部。

## 4. 技术栈参考

-   **原生开发**: 仅依赖 `jQuery` 和 `simpleParallax.js`，核心逻辑简洁高效，加载速度快。
-   **响应式适配**: 完善的媒体查询方案，确保在移动端也有极佳的显示效果。
